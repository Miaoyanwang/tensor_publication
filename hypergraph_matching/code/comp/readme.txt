### Simulation comparison between Z and W1 distance in Bernoulli matrix graph matching 

- degree_profile-master: Original codes provided by the authors of Ding et. al (2018); downloaded from https://github.com/xjmoffside/degree_profile.

- smooth_tv_b.m: Matlab script to calculate the Z distance between two empirical distributions.

- comp.m: Matlab script to run the simulation.

- comparison.pdf: Description and summary for the comparison experiment. 

- result.pdf: Figure 1 in comparison.pdf.
